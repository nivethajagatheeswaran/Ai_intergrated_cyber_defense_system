# AI Cyber Defence System

An AI-powered multi-layer cybersecurity monitoring system.

Modules Included:

• Spyware Detection
• Phishing Email Detection
• Trojan Detection
• Malware Detection
• USB Threat Detection
• WiFi Security Checker

Dashboard built using Streamlit.

## Run the system

Install dependencies:

pip install -r requirements.txt

Run scan engine:

python main.py

Launch dashboard:

streamlit run ui.py
